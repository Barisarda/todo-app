# To-Do App

Bu proje React ve Tailwind CSS kullanılarak geliştirilmiş bir görev takip uygulamasıdır.

## Kullanılan Teknolojiler

- React
- Vite
- Tailwind CSS
- JavaScript

## Proje Özellikleri

- Görev ekleme
- Görev listeleme
- Görev güncelleme
- Görev silme
- Tamamlandı işaretleme
- Local Storage desteği
- Responsive tasarım
